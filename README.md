# 📩 Spam Classifier (ML Project)

Machine Learning project to classify SMS messages as **Spam** or **Not Spam** using Natural Language Processing (NLP) techniques.

## 🚀 Features

* Text preprocessing (lowercasing, punctuation removal, stopwords removal)
* TF-IDF vectorization
* Model training using machine learning algorithms
* Evaluation using accuracy, precision, recall, and F1-score
* User input prediction
* 
## 🧠 Technologies Used

* Python
* Pandas
* NLTK
* Scikit-learn
* Google Colab

## ⚙️ How It Works

1. Load SMS dataset
2. Clean and preprocess text
3. Convert text into numerical features (TF-IDF)
4. Train ML models
5. Evaluate performance
6. Predict whether a message is spam or not

## 📊 Models Used

* Naive Bayes
* Logistic Regression
